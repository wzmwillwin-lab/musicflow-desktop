# 灵动音 MusicFlow v0.1.1

这是灵动音 MusicFlow 的第二个公开测试版。

## 本版更新

- 新增空闲闪电动效。
- 增强浏览器音乐页支持，目标覆盖 Chrome / Edge 中支持 Windows 系统媒体控制的网页播放器。
- 新增红色未读更新角标，用于提示可用更新。
- 继续保留桌面悬浮控制条、封面、进度、声波动效和播放控制。

## 下载

请下载：

- `MusicControllerPortable-v0.1.1.zip`
- `MusicController-version.json`
- `SHA256SUMS.txt`

说明：当前 beta 包内部文件名仍为 `MusicController`，后续版本会统一改为 `MusicFlow`。

## 自动更新

`MusicController-version.json` 已更新为 `0.1.1`，zip 内部的更新 manifest 地址也指向本次 GitHub Release。

同时，旧版 `v0.1.0-beta` 的 manifest 会被更新为指向 `v0.1.1`，方便已经安装旧版的用户检查到新版本。

## 已知问题

- 部分播放器可能无法显示封面或进度。
- 多屏和高 DPI 场景还需要更多测试。
- 当前安装包是绿色版，不是正式安装程序。
